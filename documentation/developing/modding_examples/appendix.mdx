## Background

While the modding example pages (e.g. [Blindfold Assist](./blindfold_assist), [Checkpoint Randomzier](./checkpoint_randomizer.mdx)) give you a deep-dive into specific mods that have been built, you might just want to quickly find examples of how to approach various mods to OpenGOAL - if so, this page is for you!

Not sure where to look? Each section has tags to help find what you're looking for - try using `ctrl+F`!
